## Reproducibility Goals

A reader should be able to:

1. inspect unit and convention choices
2. compute discretized Euclidean actions
3. compare sample path weights
4. run a teaching-scale Euclidean path Monte Carlo sampler
5. evaluate finite-dimensional Gaussian integral diagnostics
6. generate correlation-function scaffolds
7. inspect lattice scalar-field action components
8. simulate stochastic paths
9. store action, discretization, source, and simulation metadata in SQL
10. extend examples into richer functional-method workflows
