# Reproducibility Notes

## Data

Primary sample datasets:

- data/gravity_constants.csv
- data/mass_examples.csv
- data/schwarzschild_radius_sample.csv
- data/spacetime_model_metadata.csv
- data/source_metadata.csv

## Reproducibility Goals

A reader should be able to:

1. inspect the constants used in Schwarzschild calculations
2. compute Schwarzschild radii for selected masses
3. compute clock-rate factors as a function of radius
4. compare weak-field and strong-field behavior
5. generate toy gravitational-wave strain series
6. store model assumptions and source metadata in SQL
7. extend the examples into richer general-relativity workflows
