# Reproducibility Notes

## Data

Primary sample datasets:

- data/cosmic_inventory.csv
- data/bsm_candidate_classes.csv
- data/toy_constraint_points.csv

## Reproducibility Goals

A reader should be able to:

1. inspect broad BSM motivation data
2. compute schematic relic-density scaling
3. run a dark-sector parameter scan
4. summarize cosmic composition
5. store model and source metadata
6. run a toy likelihood-style constraint
7. extend examples into richer BSM workflows

## Possible Extensions

Future expansions could include:

- Boltzmann-equation freeze-out solver
- Bayesian parameter inference
- likelihood combination
- direct-detection exclusion curves
- collider missing-energy toy models
- neutrino-mass matrix examples
- axion misalignment scaling
- dark-photon kinetic-mixing parameter maps
- model-card metadata for BSM candidates
