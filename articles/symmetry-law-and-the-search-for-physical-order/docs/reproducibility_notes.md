# Reproducibility Notes

## Data

Primary sample datasets:

- data/reflection_symmetry_sample.csv
- data/symmetry_breaking_potential_sample.csv
- data/conservation_check_sample.csv
- data/symmetry_metadata.csv
- data/standard_model_symmetry_sample.csv

## Reproducibility Goals

A reader should be able to:

1. inspect reflection-invariant and broken-symmetry data
2. compute numerical invariance checks
3. evaluate harmonic-oscillator energy conservation
4. scan double-well potentials for minima
5. inspect simple gauge-phase invariance
6. store symmetry concepts and source notes in SQL
7. extend examples into richer theoretical and computational workflows
