# What Is Physics?

This folder supports the article **What Is Physics?** in the Physics knowledge series.

The article introduces physics as a discipline of measurement, mathematical law, experimental inference, numerical simulation, uncertainty analysis, and reproducible computation.

## Repository Purpose

This article folder provides advanced research-style computational scaffolding for extending the article's selected examples into a broader reproducible workflow.

## Included Materials

- Python numerical modeling and residual analysis
- Python parameter sweep for pendulum scaling
- R measurement uncertainty workflow
- Julia Runge-Kutta simulation of a pendulum
- C++ symplectic Euler simulation
- Fortran velocity-Verlet harmonic oscillator
- SQL experiment and simulation metadata schema
- Rust command-line utility for pendulum period calculations
- C instrumentation-style period estimator
- sample pendulum measurement data
- setup, methodology, and reproducibility documentation

## Physical Model

The main teaching system is the simple pendulum.

Small-angle period approximation:

T = 2π sqrt(L / g)

where:

- T = period in seconds
- L = pendulum length in meters
- g = gravitational acceleration in meters per second squared
