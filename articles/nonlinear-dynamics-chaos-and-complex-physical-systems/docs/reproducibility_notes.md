## Reproducibility Goals

A reader should be able to:

1. inspect nonlinear model parameters
2. iterate the logistic map
3. summarize long-run map behavior
4. generate bifurcation data
5. classify basic fixed-point stability
6. integrate the Lorenz system
7. compare nearby trajectories
8. estimate Lyapunov-style sensitivity diagnostics
9. store assumptions, parameters, attractor notes, and sources in SQL
10. extend examples into richer nonlinear-dynamics workflows
