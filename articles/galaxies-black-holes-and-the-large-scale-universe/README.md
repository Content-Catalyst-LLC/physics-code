# Galaxies, Black Holes, and the Large-Scale Universe

This folder supports the Physics knowledge-series article **Galaxies, Black Holes, and the Large-Scale Universe**.

The article examines galaxies as gravitational systems, black holes as extreme objects in galactic evolution, and large-scale structure as the cosmic web shaped by gravity, dark matter, and expansion.

## Repository Purpose

This folder provides advanced research-style computational scaffolding for extending the article's selected examples into reproducible astrophysics workflows.

## Included Materials

- Python workflows for enclosed mass, rotation curves, Schwarzschild radii, and Hubble-style scaling
- R workflows for schematic rotation curves, redshift summaries, and survey-style tabulation
- Julia orbital-motion scaffold
- C++ rotation-curve parameter sweep
- Fortran Schwarzschild-radius table generation
- SQL metadata for galaxies, black holes, large-scale structures, cosmic-web environments, model assumptions, and source notes
- Rust command-line utility for Schwarzschild radius and enclosed-mass calculations
- C low-level Hubble and mass-estimation example
- reproducible sample datasets
- setup, methodology, and reproducibility documentation

## Core Relations

Circular velocity and enclosed mass:

v(r)^2 = G M(r) / r

Enclosed mass:

M(r) = v(r)^2 r / G

Schwarzschild radius:

r_s = 2GM / c^2

Low-redshift Hubble relation:

v = H0 d

Redshift and wavelength:

1 + z = lambda_obs / lambda_emit
