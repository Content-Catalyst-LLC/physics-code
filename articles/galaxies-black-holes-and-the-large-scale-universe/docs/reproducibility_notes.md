# Reproducibility Notes

## Data

Primary sample datasets:

- data/rotation_curve_sample.csv
- data/black_hole_mass_sample.csv
- data/redshift_sample.csv
- data/cosmic_web_environment_sample.csv

## Reproducibility Goals

A reader should be able to:

1. inspect schematic rotation-curve data
2. compute enclosed mass from velocity and radius
3. compute Schwarzschild radii from black-hole masses
4. summarize redshift and scale-factor values
5. store cosmic-web environment metadata
6. extend examples into richer galaxy and large-scale-structure workflows

## Possible Extensions

Future expansions could include:

- NFW halo profiles
- baryonic disk plus halo decomposition
- Tully-Fisher relation examples
- galaxy merger tree metadata
- black-hole accretion efficiency calculations
- AGN luminosity and Eddington ratio examples
- DESI-style redshift catalog summaries
- cosmic-web environment classification tables
- gravitational lensing mass scaffolds
