# Cosmology and the History of the Universe

This folder supports the Physics knowledge-series article **Cosmology and the History of the Universe**.

The article examines the universe as an evolving physical system, including expansion, redshift, recombination, the cosmic microwave background, dark matter, dark energy, structure formation, and observational cosmology.

## Repository Purpose

This folder provides advanced research-style computational scaffolding for extending the article's selected examples into reproducible cosmology workflows.

## Included Materials

- Python workflows for Hubble-law modeling, redshift-scale-factor conversion, radiation-temperature scaling, and simplified ΛCDM expansion
- R workflows for redshift summaries, Hubble relation tables, and observational comparison
- Julia scale-factor and expansion-rate scaffolding
- C++ parameter sweep for simplified Hubble and expansion models
- Fortran radiation-temperature table generation
- SQL metadata for cosmic eras, cosmological parameters, source notes, and model runs
- Rust command-line utility for redshift-scale-factor conversion
- C low-level Hubble table example
- reproducible sample datasets
- setup, methodology, and reproducibility documentation

## Core Relations

Low-redshift Hubble relation:

v = H0 * d

Redshift and wavelength:

1 + z = lambda_obs / lambda_emit

Redshift and scale factor:

a = 1 / (1 + z)

Radiation temperature scaling:

T(z) = T0 * (1 + z)

Simplified flat ΛCDM expansion:

H(z) = H0 * sqrt(Omega_m(1+z)^3 + Omega_r(1+z)^4 + Omega_Lambda)
