# Reproducibility Notes

## Data

Primary sample datasets:

- data/cosmic_era_timeline.csv
- data/hubble_reference_points.csv
- data/cosmological_parameters_sample.csv

## Reproducibility Goals

A reader should be able to:

1. inspect major cosmic-era metadata
2. compute low-redshift Hubble velocities
3. convert redshift to scale factor
4. compute radiation temperature scaling
5. evaluate a simplified ΛCDM expansion function
6. store model and source metadata
7. extend the examples into richer cosmological workflows

## Possible Extensions

Future expansions could include:

- lookback-time integrals
- luminosity and angular-diameter distance calculations
- BAO standard-ruler examples
- supernova Hubble diagram scaffolding
- CMB temperature and anisotropy metadata
- Hubble tension comparison tables
- dynamical dark-energy parameterizations
- cosmological parameter inference notebooks
