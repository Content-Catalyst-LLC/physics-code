# Reproducibility Notes

## Data

Primary sample datasets:

- data/fermion_mass_sample.csv
- data/standard_model_particles.csv
- data/gauge_sector_metadata.csv
- data/higgs_sector_sample.csv
- data/source_metadata.csv

## Reproducibility Goals

A reader should be able to:

1. inspect sample Standard Model particle metadata
2. compute schematic Yukawa couplings from fermion masses
3. summarize the hierarchy across selected fermions
4. scan a schematic Higgs-like potential for minima
5. evaluate a toy running-coupling model
6. store field, particle, gauge, and source metadata in SQL
7. extend examples into richer Standard Model education workflows
