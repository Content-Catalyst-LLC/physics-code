# Quantum Fields, Particles, and the Standard Model

This folder supports the Physics knowledge-series article **Quantum Fields, Particles, and the Standard Model**.

The article examines quantum field theory, particles as field excitations, gauge symmetry, the Standard Model gauge group, matter fields, gauge bosons, electroweak symmetry breaking, the Higgs field, Yukawa couplings, renormalization, precision testing, and open questions beyond the Standard Model.

## Repository Purpose

This folder provides advanced research-style computational scaffolding for extending the article's selected examples into reproducible particle-physics and Standard Model workflows.

## Included Materials

- Python workflows for Yukawa coupling calculations, Higgs-potential scans, running-coupling toy models, and particle metadata summaries
- R workflows for fermion-mass hierarchy, Yukawa summaries, and Standard Model particle taxonomy
- Julia scalar-potential and Yukawa scaffold
- C++ running-coupling toy-model parameter sweep
- Fortran Yukawa coupling table generation
- SQL metadata for particles, fields, gauge sectors, interactions, Higgs-sector quantities, model assumptions, source notes, and simulation runs
- Rust command-line utility for Yukawa coupling calculation
- C low-level Higgs-potential table example
- reproducible sample datasets
- setup, methodology, and reproducibility documentation

## Core Relations

Standard Model gauge group:

SU(3)_C x SU(2)_L x U(1)_Y

Klein-Gordon equation:

(partial_t^2 - nabla^2 + m^2) phi = 0

Dirac equation:

(i gamma^mu partial_mu - m) psi = 0

Covariant derivative, schematic Abelian form:

D_mu = partial_mu + i g A_mu

Higgs potential, schematic scalar form:

V(phi) = mu2 phi^2 + lambda phi^4

Yukawa mass relation:

m_f = y_f v / sqrt(2)

Yukawa coupling:

y_f = sqrt(2) m_f / v
