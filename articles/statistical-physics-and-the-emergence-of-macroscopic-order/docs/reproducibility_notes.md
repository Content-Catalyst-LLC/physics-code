## Reproducibility Goals

A reader should be able to:

1. inspect constants used in statistical physics examples
2. compute exact two-state macrostate distributions
3. evaluate Boltzmann weights and partition functions
4. compare exact probability with Monte Carlo sampling
5. summarize fluctuation scaling across system size
6. explore an Ising-style lattice scaffold
7. store ensembles, state models, constants, and assumptions in SQL
8. extend the examples into richer many-particle workflows

## Possible Extensions
