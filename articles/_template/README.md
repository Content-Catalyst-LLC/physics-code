# Article Template

Use this folder as the starting structure for each Physics article.

Recommended article folders:

- `python/` for numerical modeling, symbolic math, simulation, plotting, and notebooks.
- `r/` for statistical analysis, uncertainty, regression, and measurement error.
- `julia/` for high-performance scientific computing and differential equations.
- `cpp/` for performance-critical simulation examples.
- `fortran/` for classic numerical physics and HPC-oriented examples.
- `sql/` for experiment tables, simulation metadata, and parameter sweeps.
- `rust/` for safe utilities and computational infrastructure.
- `c/` for embedded measurement, instrumentation, and microcontroller workflows.
- `docs/` for methodology, assumptions, equations, and reproducibility notes.
- `data/` for small reproducible example datasets.
- `notebooks/` for exploratory or teaching notebooks.
