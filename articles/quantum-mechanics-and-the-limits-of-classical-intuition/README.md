# Quantum Mechanics and the Limits of Classical Intuition

This folder supports the Physics knowledge-series article **Quantum Mechanics and the Limits of Classical Intuition**.

The article examines quantization, wave-particle duality, the Schrödinger equation, probability amplitudes, operators, superposition, uncertainty, measurement, and the particle-in-a-box model.

## Repository Purpose

This folder provides advanced research-style computational scaffolding for extending the article's selected examples into reproducible quantum-mechanics workflows.

## Included Materials

- Python workflows for particle-in-a-box eigenstates, expectation values, uncertainty estimates, finite-difference Hamiltonians, and Gaussian wave packets
- R workflows for probability-density summaries and measurement-style histogram comparison
- Julia finite-difference Hamiltonian scaffold
- C++ particle-in-a-box energy table generation
- Fortran energy-level table generation
- SQL metadata for simulations, operators, eigenstates, measurement samples, constants, source notes, and model assumptions
- Rust command-line utility for particle-in-a-box energy calculations
- C low-level energy-level table example
- reproducible sample datasets
- setup, methodology, and reproducibility documentation

## Core Relations

de Broglie wavelength:

lambda = h / p

Time-dependent Schrödinger equation:

i hbar partial_t psi = H psi

Stationary Schrödinger equation:

H psi = E psi

Born rule:

P(x) = |psi(x)|^2

Position-momentum uncertainty:

Delta x Delta p >= hbar / 2

Particle-in-a-box eigenstates:

psi_n(x) = sqrt(2/L) sin(n pi x / L)

Particle-in-a-box energies:

E_n = n^2 pi^2 hbar^2 / (2 m L^2)
