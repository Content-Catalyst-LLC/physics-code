# Reproducibility Notes

## Data

Primary sample datasets:

- data/planetary_orbits_normalized.csv
- data/free_fall_reference_table.csv

## Reproducibility Goals

A reader should be able to:

1. inspect idealized historical law models
2. compute Kepler-style scaling
3. model Galilean free fall
4. estimate log-log scaling exponents
5. run a two-body orbital simulation
6. store model and source metadata
7. extend the examples into richer numerical physics workflows

## Possible Extensions

Future expansions could include:

- real planetary orbital data
- Tycho-style observational uncertainty
- comparison of circular and elliptical approximations
- nonlinear orbital perturbations
- inverse-square force derivations
- symbolic derivation notebooks
- historical source annotation tables
- numerical method comparisons
