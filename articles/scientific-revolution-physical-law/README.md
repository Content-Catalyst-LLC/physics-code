# The Scientific Revolution and the Rise of Physical Law

This folder supports the Physics knowledge-series article **The Scientific Revolution and the Rise of Physical Law**.

The article examines how early modern natural philosophy was transformed through mathematics, experiment, instruments, astronomy, mechanics, and the rise of general physical law.

## Repository Purpose

This folder provides advanced research-style computational scaffolding for extending the article's selected examples into reproducible workflows.

## Included Materials

- Python workflows for Kepler-style scaling, Galilean free fall, and Newtonian gravity
- R log-log scaling and regression analysis
- Julia two-body orbital simulation
- C++ velocity-Verlet orbital integration
- Fortran free-fall and oscillator scaffolding
- SQL metadata for historical law models, simulation runs, parameter sweeps, and source notes
- Rust computational utility for Kepler-style orbital scaling
- C instrumentation-style timing and free-fall estimator
- reproducible sample datasets
- setup, methodology, and reproducibility documentation

## Core Relations

Galilean free fall from rest:

s = 0.5 * g * t^2

Kepler-style normalized scaling:

T = a^(3/2)

Newtonian gravitation:

F = G * m1 * m2 / r^2
