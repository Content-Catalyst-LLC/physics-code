## Reproducibility Goals

A reader should be able to:

1. inspect model and unit conventions
2. compute SSH winding numbers
3. estimate two-band Chern numbers
4. identify gap-closing parameter values
5. generate Berry-curvature grids
6. inspect Wilson-loop scaffolds
7. run toy Kitaev-chain diagnostics
8. store model, invariant, symmetry, source, and assumption metadata in SQL
9. extend examples into richer topological-matter workflows
