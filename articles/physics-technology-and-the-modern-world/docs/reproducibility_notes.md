# Reproducibility Notes

## Data

Primary sample datasets:

- data/timing_error_examples.csv
- data/technology_domains.csv
- data/sensor_voltage_samples.csv

## Reproducibility Goals

A reader should be able to:

1. inspect technology-domain data
2. convert timing uncertainty into distance uncertainty
3. compute power from energy and time
4. run semiconductor-style RC delay parameter sweeps
5. simulate one-dimensional heat diffusion
6. store model and source metadata
7. extend examples into richer infrastructure and device-modeling workflows

## Possible Extensions

Future expansions could include:

- GNSS error budgets
- atomic clock stability examples
- semiconductor yield metadata
- photonic link budget calculations
- thermal management models for chips
- MRI signal toy models
- energy-storage and power-density comparisons
- uncertainty propagation for sensor systems
