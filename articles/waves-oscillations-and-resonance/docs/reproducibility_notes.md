## Reproducibility Goals

A reader should be able to:

1. inspect oscillator parameters
2. compute natural frequency, damping rate, and quality factor
3. generate resonance curves
4. simulate damped and driven oscillator motion
5. compute standing-wave frequencies
6. compute wave speed, frequency, and wavelength relationships
7. run a finite-difference wave-equation scaffold
8. decompose a synthetic signal into component frequencies
9. store assumptions, wave systems, modes, and sources in SQL
10. extend examples into richer wave-physics workflows
